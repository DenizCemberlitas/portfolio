Portfolio Website

Meine eigene Portfolio-Seite – gebaut mit HTML, CSS und JavaScript.

Das Projekt

Diese Portfolio-Seite habe ich von Grund auf selbst gebaut um meine Projekte und mein Homelab zu präsentieren. Es war gleichzeitig mein erstes Projekt bei dem ich wirklich verstanden habe wie HTML, CSS und JavaScript zusammenspielen.

Features

Die Seite lädt automatisch meine Top 6 GitHub Repositories über die GitHub API und zeigt sie als Karten an. Außerdem gibt es einen selbst gebauten Contribution Graph mit Hover-Tooltip, einen Homelab Blog und eine Projektdokumentation.

Was ich gelernt habe

Ich habe gelernt wie man mit fetch() externe APIs anspricht, wie CSS Grid und Flexbox für Layouts funktionieren, wie man mit Media Queries responsive Design umsetzt und wie man eine Seite mit GitHub Pages deployed.

Tech Stack

HTML · CSS · JavaScript · GitHub API · GitHub Pages
